# Examples

Runnable demonstrations of the golang-ui library.

```bash
go run ./examples/signals    # Reactive primitives: Signal, Computed, Subscribe, Batch
go run ./examples/counter    # Counter with derived values
go run ./examples/forms      # Reactive form with validation
go run ./examples/shopping   # Shopping cart with discounts and tax
go run ./examples/dashboard  # Dashboard with filtering and metrics
go run ./examples/theme      # Design tokens: colors, spacing, radius, typography
```
